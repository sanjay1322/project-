AddFont

AddFont(string family [, string style [, string file [, string dir]]])

Açıklama

Bir TrueType, OpenType veya Type1 yazı tipini içe aktarır ve kullanılabilir hale getirir. İlk olarak MakeFont yardımcı programıyla bir yazı tipi tanım dosyası oluşturmak gereklidir.

Tanım dosyası (ve yerleştirme durumunda yazı tipi dosyası kendisi) aşağıdaki yerlerde bulunmalıdır:
  • 4. parametre tarafından belirtilen dizin (eğer bu parametre ayarlanmışsa)
  • FPDF_FONTPATH sabiti tarafından belirtilen dizin (bu sabit tanımlıysa)
  • fpdf.php ile aynı dizinde bulunan font dizini
Dosya bulunamazsa, "Could not include font definition file" hatası oluşur.

Parametreler

family
Yazı tipi ailesi. İsim keyfi olarak seçilebilir. Eğer standart bir aile adıysa, ilgili yazı tipini geçersiz kılar.
style
Yazı tipi stili. Olası değerler (büyük-küçük harf duyarsız):
  • boş dize: düz
  • B: kalın
  • I: italik
  • BI veya IB: kalın italik
Varsayılan değer düzdür.
file
Yazı tipi tanım dosyasının adı.
Varsayılan olarak, aile ve stil adından küçük harfle ve boşluk olmadan oluşturulur.
dir
Tanım dosyasını yüklemek için dizin.
Belirtilmezse, varsayılan dizin kullanılır.

Örnek

$pdf->AddFont('Comic','I');
ile aşağıdaki aynıdır: 
$pdf->AddFont('Comic','I','comici.php');

Bunlara da bakın

SetFont().
