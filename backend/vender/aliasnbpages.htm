AliasNbPages

AliasNbPages([string alias])

Açıklama

Toplam sayfa sayısı için bir takma ad tanımlar. Belge kapatıldığında bu takma adın yerine geçirilecektir.

Parametreler

alias
Takma isim. Geçerli değer: {nb}.

Örnek

class PDF extends FPDF
{
    function Footer()
    {
        // 1.5 cm alt kenardan başla
        $this->SetY(-15);
        // Arial italic 8'i seç
        $this->SetFont('Arial', 'I', 8);
        // Mevcut ve toplam sayfa numaralarını yazdır
        $this->Cell(0, 10, 'Sayfa '.$this->PageNo().'/{nb}', 0, 0, 'C');
    }
}

$pdf = new PDF();
$pdf->AliasNbPages();
