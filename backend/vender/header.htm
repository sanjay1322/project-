Header()

Tanımlama

Bu yöntem sayfa başlığını görüntülemek için kullanılır. Otomatik olarak AddPage() tarafından çağrılır ve uygulama tarafından doğrudan çağrılmamalıdır. FPDF'deki uygulama boştur, bu nedenle belirli bir işlem yapmak istiyorsanız bunu alt sınıf olarak tanımlamalı ve yöntemi geçersiz kılmalısınız.

Örnek

class PDF extends FPDF
{
    function Header()
    {
        // Arial bold 15'i seç
        $this->SetFont('Arial', 'B', 15);
        // Sağa taşı
        $this->Cell(80);
        // Çerçeveli başlık
        $this->Cell(30, 10, 'Başlık', 1, 0, 'C');
        // Satır sonu
        $this->Ln(20);
    }
}
