Footer

Footer()

Açıklama

Bu yöntem sayfa alt bilgisini görüntülemek için kullanılır. Otomatik olarak AddPage() ve Close() tarafından çağrılır ve uygulama tarafından doğrudan çağrılmamalıdır. FPDF'deki uygulama boştur, bu nedenle belirli bir işlem yapmak istiyorsanız bunu alt sınıf olarak tanımlamalı ve yöntemi geçersiz kılmalısınız.

Örnek

class PDF extends FPDF
{
    function Footer()
    {
        // 1.5 cm alt kenardan başla
        $this->SetY(-15);
        // Arial italic 8'i seç
        $this->SetFont('Arial', 'I', 8);
        // Ortalanmış sayfa numarasını yazdır
        $this->Cell(0, 10, 'Sayfa '.$this->PageNo(), 0, 0, 'C');
    }
}
